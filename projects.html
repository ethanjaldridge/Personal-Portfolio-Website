Workout Advice Generator

  As part of the codecademy curriculum, the "Mixed Messages" assignment was my first project using JavaScript. I chose to create a Workout Advice Generator. Each time a user runs the program, it provides a randomized piece of workout advice. The workout advice is created from 3 pieces of data: a goal, something to start doing, and something to stop doing. Each piece of data only has a few possible outputs, so the program is fairly limited. Nonetheless, this is my first self-written JavaScript program.
